# Portal de Benefícios Raguife · v7 Separado

Arquivos:
- index.html: portal do funcionário, otimizado para celular.
- rh.html: painel web do RH/DP.

1. Suba os dois arquivos na raiz do repositório.
2. O QR Code deve apontar para index.html.
3. O RH/DP acessa rh.html.
